Academic & Personal Projects

LLM Inference Optimization

Independent Research (Aligned with Systems + AI Optimization)

Tech Stack: Python, PyTorch, Hugging Face Transformers, Profiling Tools

Focus: Efficient AI Systems, LLM Optimization

LLM Inference Optimization is a research-oriented prototype focused on improving the latency and memory efficiency of transformer-based large language models during inference. The project investigates how system-level optimizations such as KV cache reuse and model quantization impact performance under varying workload conditions, particularly in long-context scenarios.

  • Implemented KV cache mechanisms to eliminate redundant attention computations during sequential token generation
  • Profiled system behavior to identify memory-bound bottlenecks in transformer inference pipelines
  • Analyzed trade-offs between model efficiency and output quality under different optimization strategies

Research Motivation: How system-level optimizations can improve scalability and real-time performance of large language models in resource-constrained and latency-sensitive environments

Compass AI — Agentic Monitoring and Governance for AI Systems

Associated with: Johnson Controls

Tech Stack: Python, Large Language Models, Vector Databases, Cloud Services, Distributed Logging & Monitoring

Focus: Agentic AI, ML Systems, Reliability, AI Safety

Compass AI is a research-oriented prototype designed to address the lack of observability, reliability, and governance in large-scale AI systems, particularly those integrating large language models (LLMs) into production workflows.The system introduces an agentic supervisory layer that continuously monitors AI behavior, evaluates output quality, and autonomously triggers corrective actions such as rerouting, retries, or fallback strategies.

  • Designed autonomous observer, evaluator, and decision-making agents
  • Implemented closed-loop control for AI system reliability
  • Explored metrics for hallucination detection, drift, and policy violations
  • Integrated memory and logging layers for longitudinal behavior analysis

Research Motivation: How agentic control loops can improve robustness, trust, and fault tolerance in LLM-driven systems.

Decentralized Audit and Provenance System for Intelligent Services

Associated with: DBATU/Pre-Final Year

Tech Stack: Blockchain (Permissioned Ledger), Smart Contracts, Cryptographic Hashing, Off-chain Storage Systems

Focus: Blockchain Systems, Distributed Ledgers, System Integrity

This project investigates the use of blockchain-based architectures to provide auditability, provenance, and tamper-resistance for decisions generated by distributed software systems. The primary objective is to ensure system-level transparency and accountability in environments where decisions must be verifiable over time.

  • Designed a permissioned blockchain architecture for enterprise environments
  • Implemented immutable logging of system events and decision metadata
  • Used smart contracts to enforce validation and access-control policies
  • Enabled off-chain storage with on-chain integrity verification

Research Motivation: Exploring how decentralized ledgers can improve trust, accountability, and fault diagnosis in large-scale distributed systems without introducing significant performance overhead.
